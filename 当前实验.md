# Step 6: MVP evaluation

重要修正：

TIP 不能简化成 KL-only。TIP 的核心是 student entropy 与 teacher-student divergence 的二维 taxonomy，完整 selection baseline 应包含：

* entropy-only
* divergence-only
* TIP Soft-OR: entropy + divergence

因此，下面旧的 `TIP` 表述应理解为 `divergence-only baseline`。正式结果需要补 entropy-aware baseline。

不用训练。

直接看：
